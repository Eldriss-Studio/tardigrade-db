Vamana graph index (DiskANN-style, NOT HNSW).

Single-layer graph for approximate nearest neighbor search over key vectors. Uses greedy search with a medoid start node. Suitable for cold-path retrieval at >10K blocks where brute-force becomes too slow.

Key parameters:

  • R (max_degree): maximum out-degree per node. Higher = better recall, more memory.
  • L (search_list_size): beam width during greedy search. Higher = better recall, slower.
