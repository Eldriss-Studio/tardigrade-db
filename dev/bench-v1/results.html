LoCoMo Ceiling Analysis

68.2% is the model capability limit for Qwen3-0.6B on LoCoMo, not a retrieval failure.

Six refinement techniques were tested on LoCoMo (whitening, cross-encoder reranking, keyword/embedding/generative RLS variants) — all produced 0% improvement over the baseline. This is consistent with the ICLR 2026 LIMIT paper, which proves a theoretical performance ceiling for small models on long-context memory tasks correlated with model size.

Root cause: LoCoMo queries use abstract vocabulary ("what outdoor activities", "science-related hobby") while stored conversations use concrete vocabulary ("marathon", "ecology"). The vocabulary gap is not bridgeable by retrieval tuning alone — it requires a model that can map between abstract and concrete language. For Qwen3-0.6B, 68.2% is the hard ceiling until a larger model or vocabulary-bridge strategy (e.g. LLMAgentReformulationStrategy) is validated.
