DIM_IDX
Index within the header where the vector dimension is stored. Survives Q4.
HEADER_SENTINEL
Sentinel value at position 0 marking a per-token encoded key.
HEADER_SIZE
Header spans two Q4 groups (64 floats) so the sentinel can dominate its own group (-1e9 abs_max) without crushing the data values that follow.
N_TOKENS_IDX
Index within the header reserved for token count. Informational only — see HEADER_SIZE for why this field is not trusted on read.
